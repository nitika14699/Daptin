Types
===

Basic types

- Id
- Alias
- Date
- Time
- Day
- Month
- Year
- Minute
- Hour
- Email
- Name
- Value
- TrueFalse
- DateTime
- Location (lat/long)
- Color
- Measurement
- Label
- Content
- File
- Url
- Image


Collection types

- Pair = (x,y) where x,y is from Basic types
- Triplet = (x, y, z) where x,y,z is from basic types


- Set = []{x1, x2, x3..} where x1, x2, x2 are of same type and one of the basic type
- OrderedList = same as Set, but have ordering
